# Title: Testcase 4

1. Navigate to the products page with filters that return no results
2. Wait for the system to fetch products from the API
3. Observe the message displayed when no products are available

## Expected Results
- API call completes without errors
- No product items are rendered on the screen
- "No products found" message is displayed to the user
- Message is clearly visible and easy to understand
