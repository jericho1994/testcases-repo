# Title: Submit a message through the contact form
## User Story:

As a website visitor  
I want to send a message with my name, email, subject, and message content  
So that I can contact the website administrator  

## API Endpoint: POST https://api.practicesoftwaretesting.com/messages

## Request Body (JSON): {
    "name": "awdawd awdaw",
    "subject": "webmaster",
    "message": "adwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdadadwawdawdawdadawdad",
    "email": "dawdawd@yahoo.com"
}

## Acceptance Criteria:
* The system sends a POST request to /messages
* The request body must include:
  * name
  * subject
  * message
  * email
* All fields are required.
* The email field must follow a valid email format.
* If the request is valid:
  * The API returns a success status (e.g., 200 OK or 201 Created).
* If required fields are missing or invalid:
  * The API returns an appropriate error (e.g., 400 Bad Request).
