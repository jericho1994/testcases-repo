# Title: View products filtered by price range and rental status
## User Story:

As a customer
I want to view products priced between $1 and $100 that are not rental items
So that I can browse affordable products available for purchase

## API Endpoint: GET https://api.practicesoftwaretesting.com/products?page=1&between=price,1,100&is_rental=false

## Acceptance Criteria:
* The system sends a GET request to: /products?page=1&between=price,1,100&is_rental=false
* The response status code should be 200 OK.
* The response should return a list of products.
* All returned products must:
  * Have a price between 1 and 100
  * Have "is_rental": false
* Pagination should return results for page 1 only.
